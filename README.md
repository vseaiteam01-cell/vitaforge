# 💪 Fit Tracker — Telegram Mini App

Мини-апп для трекинга силовых тренировок: вносишь текущие веса → апп считает прогрессию,
прогнозирует рост силы (оценка 1ПМ) и сам ставит в **календарь**, в какой день добавить вес
(«16 июл — жим +2,5 кг»). Плюс раздел «Тело»: замеры, документы и медицинские анализы.

Работает **полностью на клиенте** (HTML/CSS/JS, без бэкенда). Данные хранятся на устройстве
(`localStorage`). Открывается и в обычном браузере, и внутри Telegram.

## Что внутри

- **План** — твой норматив (3 дня: грудь/руки/плечи, спина, ноги). По каждому упражнению:
  рабочий вес × повторы × подходы, оценка 1ПМ, кнопки «Лог» (записать подход) и «✏️» (настроить).
- **Прогресс** — силовой потенциал: текущий вес → прогноз через N недель по каждому упражнению,
  спарклайны, суммарный прогноз роста. Тап по строке → график **план/факт** + история.
- **Календарь** — расписание прибавок веса, привязанное к твоим дням тренировок.
  Двойная прогрессия: добрал верх диапазона повторов → апп планирует +шаг.
- **Тело** — ручные замеры (вес / % жира / мышцы), ссылка на твой youjiu-отчёт,
  загрузка документов и анализов (хранятся локально; авто-разбор ИИ — следующий этап).

Профиль (⚙️ вверху): вес тела, горизонт прогноза, дни недели тренировок, экспорт JSON, сброс.

## Как посмотреть локально

Любой статический сервер из папки проекта:

```powershell
# Python
python -m http.server 8080
# или Node
npx serve .
```

Открой `http://localhost:8080`. (Открывать `index.html` файлом напрямую тоже можно,
но через сервер ближе к боевым условиям.)

## Как превратить в настоящий Telegram Mini App

1. **Захостить папку по HTTPS.** Telegram пускает мини-аппы только по https.
   Бесплатные варианты: GitHub Pages, Cloudflare Pages, Netlify, Vercel.
   Залей `index.html`, `styles.css`, `app.js`, `data.js` в репозиторий → включи Pages →
   получишь ссылку вида `https://<user>.github.io/fit-tracker/`.
2. **Создать бота** у [@BotFather](https://t.me/BotFather): `/newbot`.
3. **Привязать мини-апп:** в BotFather `/newapp` (или Bot Settings → Menu Button → Configure menu button)
   и указать URL из п.1. Можно также `/setmenubutton`.
4. Открываешь бота → кнопка меню запускает апп прямо в Телеге, с твоей темой оформления.

> SDK Telegram (`telegram-web-app.js`) уже подключён — апп забирает тему, разворачивается
> на весь экран, использует нативную кнопку «Назад» и тактильный отклик.

## Следующие этапы (когда захочешь)

- **Облачная синхронизация** между устройствами — нужен лёгкий бэкенд (например, бот + БД
  или Cloudflare Workers + KV) вместо `localStorage`.
- **Авто-разбор анализов и состава тела** — загрузка PDF/InBody → ИИ извлекает показатели,
  связывает с прогрессом (требует бэкенда с доступом к модели).
- **Импорт youjiu** — отчёт защищён токеном (SPA), автоподтягивание данных тоже про бэкенд.

## Модель прогрессии (как считает)

- **Оценка 1ПМ** (формула Эпли): `1ПМ = вес × (1 + повторы / 30)` — позволяет сравнивать силу
  при разном числе повторов.
- **Двойная прогрессия:** растёшь в повторах внутри диапазона `range`; добрал верх — пора +`inc` кг.
- **Календарь/прогноз:** каждое упражнение прибавляет `inc` кг раз в `stepWeeks` недель,
  прибавка ставится на реальный день тренировки соответствующей группы.
  Всё это настраивается по каждому упражнению (✏️) и в профиле (⚙️).
